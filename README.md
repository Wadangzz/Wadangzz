## Hi there 👋
자동화 시스템과 스마트 팩토리에 관심 많은 **엔지니어**입니다.

현장의 움직임을 제어하고, 데이터를 흐르게 하며, 디지털로 시각화하는 것에 흥미를 느낍니다.  
PLC, 로봇, 센서, 머신비전, 그리고 그것들을 하나로 묶어주는 통신과 인터페이스 설계에 집중하고 있습니다.
