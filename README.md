## 👋 Hi There

📈 데이터를 수집하고 분석하며  
🖥️ 디지털로 시각화하는 일에 흥미를 느끼고 있습니다.  

  
🤖 PLC, 로봇, 센서, 머신비전 기술을 하나씩 익히고 있으며  
🔗 이들을 통합하는 통신 시스템과  
🌐 Digital Twin 구축도 꾸준히 도전하고 있습니다.  

  
📚 작은 경험을 차곡차곡 쌓아가며,
